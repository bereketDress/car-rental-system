# Car Rental Management System (CRMS)

## Problem Statement
DriveEasy Rentals currently manages vehicle bookings and customer records manually across multiple branches, leading to inefficiencies such as booking conflicts, poor tracking, and slow operations. This system aims to automate and centralize rental management.

## Features
- Vehicle management across branches  
- Customer registration  
- Reservation system  
- Rental tracking and billing  
- Admin dashboard and reporting  

## Tech Stack
- Java / Spring Boot  
- PostgreSQL  
- React.jsx
