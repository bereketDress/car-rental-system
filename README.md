# Car Rental Management System

## Project Description
This project is a full-stack web application developed to automate and manage car rental operations. The system allows customers to search, reserve, and rent vehicles online while administrators manage vehicles, customers, and rental transactions.

## Problem Statement
Many car rental businesses still manage reservations and customer records manually, leading to booking conflicts, inefficient operations, and poor customer experience. This project aims to provide a centralized and automated solution for managing rental services efficiently.

## Features
- Vehicle booking
- Customer registration
- Online payments
- Rental tracking
- Admin dashboard

## Technologies
- Java / Spring Boot
- postgres
- React.js
- GitHub
