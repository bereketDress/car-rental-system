Customer Stories:
As a customer, I want to register, so I can use the system
As a customer, I want to search vehicles, so I can choose a car
As a customer, I want to make a reservation, so I can book a car
As a customer, I want to view my bookings, so I can track my rentals
👨‍💼 Staff Stories:
As staff, I want to check-out vehicles, so I can release cars to customers
As staff, I want to check-in vehicles, so I can complete rentals
As staff, I want to manage reservations, so I can control bookings
🧑‍💼 Admin Stories:
As admin, I want to manage vehicles, so I can maintain fleet data
As admin, I want to manage customers, so I can control users
As admin, I want to view reports, so I can analyze business performance
