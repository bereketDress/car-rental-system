## Customer stories

- As a customer, I want to register, so I can use the system
- As a customer, I want to search vehicles, so I can choose a car
- As a customer, I want to make a reservation, so I can book a car
- As a customer, I want to make a payment, so I can complete my booking
- As a customer, I want to view my booking history, so I can track my rentals

## Staff stories

- As a staff member, I want to manage reservations, so I can control bookings
- As a staff member, I want to check out a vehicle, so I can release a car to a customer
- As a staff member, I want to record mileage, fuel, and damage at check-out, so I have a baseline for return
- As a staff member, I want to check in a returned vehicle, so I can complete the rental
- As a staff member, I want to update the rental status, so the system reflects the current state
- As a staff member, I want to process the final payment, so the customer's bill is settled

## Admin stories

- As an admin, I want to manage vehicles, so I can maintain fleet data
- As an admin, I want to manage customers, so I can control user accounts
- As an admin, I want to manage branches, so I can keep location data up to date
- As an admin, I want to generate reports, so I can analyze business performance
